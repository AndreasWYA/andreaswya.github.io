MathAdvance

MathAdvance adalah sebuah Aplikasi Pembelajaran Matematika dengan Gamifikasi untuk Anak-anak bisa lebih tertarik dan bermotivasi untuk belajar Matematika. Appnya memiliki berbagai macam Mini Games untuk anak-anak memainnya sambil belajar dari game tersebut. Serta Mini Games tersebut terdapat berbagai macam materi dan subyek matematika yang tersedia dan juga fitur-fitur lainnya yang mereka bisa gunakan untuk menyambung ke teman-teman mereka seperti menambahkan teman dan juga pamerkan Lambang-lambang yang mereka memiliki dari memainkan aplikasi tersebut.

Game Quiz Web

Game Quiz Web adalah sebuah proyek Front-end yang telah dikerjakan untuk bisa membantu meningkatkan kemampuan programming dalam Front-end Libraries dengan penggunaan seperti Javascript. Di dalam juga ada Leaderboards untuk 2 Stages dari Quiz Game tersebut. Lalu project ini hanya bisa dijalankan melalui Localhosting contoh dengan XAMPP atau Live Server pada Visual Studio Code.
